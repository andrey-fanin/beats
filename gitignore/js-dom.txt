            DOM - Document Object Model
Узлы элементы: открывающий тег, наполнение.
Текстовые узлы: текст внутри тэга <h1> "Привет" </h1>


            DOM-события
Все действия пользователя и есть dom-события. (вся работа пользователя со страницей)


            Формочки
Кнопка в форме при нажатии по умолчанию отправляет все данные из полей и перезагружает страницу (action у формы)
