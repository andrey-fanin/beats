                SVG - масштабируемая векторная графика.
Отличие векторного от растрового: 
 Растровые - из точек. (jpeg 3байта, gif, png)
 Векторные - из фигур. Абстрактная система координат. Вес будет одинаковым при любом размере.

Преимущества:
 Чётко выглядит на любом дисплее
 Снижает количество запросов
 Легко поддаётся масштабированию
 Готов для адаптива
 Меньший размер файлов
 Удобны для анимации

XML - eXtensible Markup Language - расширяемый язык разметки. Можно самому определять теги и атрибуты.
Json пришёл на смену XML.

    <SVG xmlns="">
        <reсt x="10", y="20", width="100", 
        rx="радиус округления углов по X", ry="и по y"
        fill="закрасить объект"
        stroke="цвет границы"
        stroke-width="ширина границы"

Стили css приоритетнее атрибутов для свг.
Чем выше фигура в rect, тем ниже она в слоях.


                Фигуры
    <rect x= y= width= height=> прямоугольник
    <circle cx="50" cy="50" r="20" fill="по умолчанию чёрный">
    <ellipse cx="80" cy="80" rx="30" ry="20">
    <line x1="10" y1="20" x2="80" y2="200" без stroke="цвет" не видно!!!>
    <polyline points="10 20, 40 30, 70 100" у последнего соединения границы не будет>
    <polygon - как полилайн, но закрытая фигура.>
    <path d="M20,20 L150,150 M50,20" stroke="цвет" - движение абстрактного пера, для соединения последней точки с первой последнее число N"z"!!

    M - move to
    L - Line to
    Z - close path
    H, V - Horizontal / Vertical Line
    C - кривая
    Q - кривая Безье
    A - дуга 

        Подготовка к загрузке
    Чтобы внутри контейнера определить систему координат, добавь viewBox="0 0 (начало относительно правого верхнего угла) 
    (100 100)кол-во абстрактных точек для контейнера
    viewBox="0 0 100 100"
    preserveAspectRatio="align - выравнивание, type - тип"
    preserveAspectRatio="xMaxYMax(на весь размер) Middle(середина) Min (начало) / meet - фигура будет целиком, slice - обрезать, none - непропорционально"

        Группировка элементов
    <g fill="red" (если нет своих стилей, то применятся общие)> - Группировка
    
    <defs> - без вызова их не видно
        <g

    <use xlink:href="#blue-circle" x="20" y="10">
    <symbol> - сразу много элементов, внутри может быть свой viewBox

    <svg>
        <filter id="blur">
            <feGaussianBlur in="SourceGraphic" (можно отдельно на альфа-каналы, заливку, границу и т.д.) stdDeviation="3" (from 1 to 10)
            <image filter="url(#blur) width="70" height="100" xlink:href="./lol" x="0" y="0"
            
    img {
        filter: url('./filters.svg#blur')
    }

    <feColorMatrix type="saturate" values="0" - обесцветить
    <feColorMatrix type="heuRotate" values="170(deg)" - изменить цвет по углу цветового круга

                Градиенты
    <linearGragient id="1" x1="0" y1="0" x2="100%" y2="100%">
        <stop offset="0%" stop-color="#00f">
        <stop offset="100%" stop-color="$0F0"

    <text x="20" y="10">LS</text> - точка начала текста отсчитывается от левого нижнего угла надписи
        <tspan - для закрашивания всего цвета одним градиентом
